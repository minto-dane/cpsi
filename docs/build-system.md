# ビルドシステム

CPSI はビルドシステムとパッケージ管理を分離します。

## ビルドレシピ

ビルドレシピは `.cpsb` 拡張子を持ちます。

```text
.cpsb
```

`.cpsb` は CPS Build Recipe を表します。

## 役割

`.cpsb` はパッケージ作成専用です。

利用者は通常 `.cpsb` を取得しません。

## ビルドの流れ

```text
foo.cpsb
 ↓
cpsbuild build
 ↓
foo-1.0-k1-x86_64.clos
```

## 責務

`.cpsb` の責務:

- ソース取得
- パッチ適用
- ビルド
- パッケージ生成

`.clos` の責務:

- 配布
- インストール
